2D Platformer Game

Available on itch.io

This is a 2D platformer game that includes enemies and collectons. This is my first game when I just started to learn Unity.

I learned Unity following tutotials. This is my first game and it used the unity assets store's free asset "Sunnyland". The player needs to control the fox and navigate through obstacles and avoid the enemies to complete the game. It was awarding to learn how each components in game link together. I haven't been able to put my own drawings in my game but I certainly want to make a story based game with my drawings and plan to learn asprite for pixelated paintings.
