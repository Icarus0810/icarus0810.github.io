3D Snake Game

Available on itch.io

This is a 3D version of the classic snake game that I made. Making it into a 3D sphere matched the original design of a snake running endlessly in a square space.

One day I just came up with this idea of trying to replicate the snake game but in a 3D version. It was so fun designing it and finally realizing my thoughts. I learned to code a gravity attractor body in this game so that the snake body always snaps to the shpere. I also learned simple modeling in blender and texture mapping during this time. The game includes not only single but local coop and local versus mode. It was fun to play with friends. The game is available on itch.io
